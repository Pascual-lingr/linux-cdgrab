Manejo del interfaz linux-cdgrab
--------------------------------

Las opciones del programa van numeradas. Segun la tarea ha realizar,debe
 teclearse el numero correspondiente de la opcion desada.

Las principales opciones son las siguientes:

1.- GRABACION EN CD --> Para hacer grabaciones de datos,audio y cdmixto en CD
                        entre otras.

2.- GRABACION EN DVD --> Para hacer grabaciones de datos y video en DVD
                         entre otras.

3.- CREACION DE IMAGENES --> Para crear imagenes de datos en varios formatos
                             entre otras.
                             Formatos iso-9660, .img, .bin .cue y ext2.

4.- CONVERSION DE AUDIO --> Convierte formatos de audio y ripea CDs de Música.

5.- MONTAR O DESMONTAR CD-ROM --> Monta o desmonta un CD-ROM o DVD-ROM

6.- MONTAR O DESMONTAR IMAGEN --> Monta o desmonta una imagen .iso, .ext2 o .img

7.- CARGAR PARCHE ADICIONAL --> Ejecuta un parche para la aplicacion

8.- CENTRO DE CONTROL --> Configuracion de Linux-cdgrab

9.- AYUDA --> Muestra este mismo archivo de ayuda.

E o e --> Expulsa un CD o DVD de las unidades de CD-ROM o DVD-ROM

TD --> Breve menú para realizar diversas acciones sobre los archivos
       de imagen de disco creados en el directorio /tmp

G --> Gestor de archivos integrado en Linux-cdgrab.
      Permite navegar por el sistema de archivos y realizar ciertas
      acciones sobre ficheros y directorios.

A --> Creditos del programa

Opciones principales ocultas:

El script acepta 3 opciones principales que permanecen ocultas al
 menú principal.

logo --> Imprime el logo ASCII de LINUX-CDGRAB 0.5

dep --> ejecuta LINUX-CDGRAB 0.5 en modo depuracion.
deppatch --> ejecuta un parche en modo depuracion.

dep y deppatch son opciones avanzadas para observar errores de ejecucion en 
linux-cdgrab.

OTRAS ACLARACIONES SOBRE EL MANEJO DE LA APLICACION:

Linux-cdgrab pedira en ocasiones que teclee rutas de directorios
 para poder grabar sus archivos.
Debe siempre indicarse la ruta ABSOLUTA de los archivos o directorios
 que usted quiera grabar.

Un ejemplo:

En el directorio /usr/local hay una imagen ISO-9660 llamada
 "disco_para_grabar.iso".
Para grabar esta imagen debe indicarle al programa la ruta de directorios
 absoluta donde esta este almacenada.

Se debe introducir la ruta de esta forma: /usr/local/disco_para_grabar.iso

Configuracion de Linux-cdgrab:
------------------------------

-Velocidad de grabacion:

Indicar solamente un numero , por ejemplo , 16.
No se acepta ningún valor de velocidad máxima.

-Canal IDE de la grabadora:

El canal IDE/SCSI donde este conectada la grabadora.

Puedede averigurase ejecutando la siguiente orden:
cdrecord -scanbus
Un ejemplo del canal IDE : 0,0,0 o bien 0,1,0 etc.
El programa tambien admite la posibilidad de especificar el dispositivo de
 bloque que corresponde con la grabadora. Ejemplo: /dev/hdd

Debe optar por especificar el canal IDE o el dispositivo de grabacion
 segun la version de cdrecord que utilice, pero no ambos.

- Canal ATAPI

El canal IDE/ATAPI donde este conectada la grabadora.
Este parametro es necesitado por Linux-cdgrab en la grabacion de discos
 multisesion.

Su nomenclatura suele ser 0,0,0 o bien 0,1,0 o o bien 1,0,0 etc.
No configurar con el nombre de un dispositivo de bloque.

-Etiqueta para el CD o imagen de CD:

Corresponde con el nombre que quiera darle a su CD o imagen de CD.
Por ejemplo: fotografias,datos etc.

-Ruta de directorios :
La ruta de directorios donde se encuentran los datos que quiera grabar.
Por ejemplo /home/usuario/documentos
Recuerde que tiene que especificar la ruta absoluta de directorios.

Descripciones
-------------

-Imagen de CD
Una imagen de CD es un archivo que contiene la réplica exacta de un CD original.
Si se hace una imagen de un CD obtendremos en disco un archivo que contiene todos los datos que estan en el CD de origen.

Estos archivos se utilizan para grabar varios CDs iguales o para que el proceso de grabacion sea mas seguro.Si desea ver el contenido de una imagen tendra que
 montarla.

Las imagenes por ser una replica de un CD tienen su mismo sistema de ficheros
 por lo que si monta alguna no podra escribir en ella.
El sistema de ficheros ISO-9660 es de solo lectura.

--Imagen ext2 , esto crea una imagen vacia con el sistema de ficheros de Linux
  (se tendra acceso de lectura y escritura sobre este tipo de imagenes) la cual
  debe  montarse y copiarse en ella,los datos que se deseen incluir.

Estas imagenes una vez grabadas en CD no son accesibles con permisos de
 escritura.

El sistema de ficheros ext2 es el sistema nativo de Linux. Bajo este sistema
operativo el contenido de estos cds sera totalmente accesible.

En el caso de Windows o versiones de Mac OS, estos CDs no seran reconocibles.

--- Especial mencion a los niveles ISO ---

Nivel iso 1
-----------
Los nombres de archivo consisten en una seccion
y estan limitados a 8.3 caracteres.
Elija este nivel si desea tener compatibilidad de nombres
bajo el sistema operativo MS-DOS

Nivel iso 2
-----------
Los nombres de archivo consisten en una seccion

Nivel iso 3
-----------

Nivel sin restricciones.
Use este nivel si quiere compatibilidad con ficheros
grandes,en este caso >= a 2 Gigabytes.
