			LINUX - CDGRAB ver 0.5 r2
			=========================

			   20 - Febrero - 2020

LINUX-CDGRAB es un front-end en modo consola para grabar discos ópticos (CD,DVD)

La aplicación utiliza las herramientas de grabación cdrtools,cdrdao
y dvd+-rw-tools para estas funciones.

Permite también la creación de imagenes de CD y DVD, junto con conversión de
ficheros de audio, compresión de archivos o bien el montado de unidades o
imagenes de disco, entre otras opciones. Su interfaz es multilenguaje.

El programa es un conjunto de scripts de shell que actuan como un interfaz
de menus interactivos, encargados de llamar a las diferentes herramientas
de grabación junto a otras diversas existenes en nuestra instalación,
para llevar a cabo las funciones de grabación de discos ópticos, creación
de imágenes, montaje de unidades e imágenes, operaciones sobre dispositivos
de bloque, conversión de audio... etc.

Este programa ha sido diseñado únicamente para el sistema operativo GNU/Linux.
Solamente sera funcional en todas sus opciones si es utilizado bajo
este sistema.
Puede ejecutarse en otras versiones de UNIX e incluso en implementaciones
como Cygwin o el subsistema Linux de Windows 10, pero no se ha tenido en
cuenta la compatibilidad con otro UNIX que no sea Linux.

Se incluye un parche en esta versión para dar compatibilidad al etiquetado
de CD y DVD LightScribe.
El proceso de etiquetado se lleva a cabo mediante la herramienta
privativa 4L de Lacie.

Requisitos:
-----------
Las herramientas de grabación cdrtools (acepta cdrkit), cdrdao
y dvd+-rw-tools.
Compresores de archivo zip, xz, gzip, bzip2.
Conversores de sonido lame, oggenc y sox.
Dispositivos /dev/loop para montar imagenes de CD/DVD.
Utilidades estandares de Linux como tar, grep, dd, mke2fs, cat etc.

Necesitara tener todos los programas instalados en su sistema para
 que todas las opciones de la aplicacion funcionen correctamente.

Las distribuciones de Linux suelen tener estas herramientas instaladas.

Notas sobre el soporte LightScribe:

Necesaria la herramienta 4L de Lacie para el parche LightScribe.

Si usted no tiene una grabadora para etiquetar CDs o DVDs LightScribe
puede prescindir el uso de este parche.

Instalación:
------------

Descomprima el programa ejecutando:

tar -xvzf lingrXY.tgz ( X e Y corresponden a la versión del programa )

Dentro del directorio de archivos de linux-cdgrab teclee ./instalar

NOTA:

El programa se copiará en /usr/bin y podra ejecutarlo mediante la orden lcdgrab.
La instalación creara el directorio /mnt/lcdgrab,éste es usado por linux-cdgrab como punto de montaje.
El archivo lcdgrab, leeme.txt, GPL-2.0.txt, logo.txt y ayuda.txt se copiaran en /usr/share/linux-cdgrab.
Si existen parches en la capeta de instalación se copiarán en /usr/share/linux-cdgrab/parches
Si existen ficheros de idioma para los parches, seran copiados en /usr/share/linux-cdgrab/lcdgrabAPI_idiom/*/lcdgrabparches_idiom
Adicionalmente a parches se crean los siguiente directorios en /usr/share/linux-cdgrab
../lcdgrabAPP_PI
../lcdgrabCMD_API
../lcdgrabMenu_API
../lcdgrabAPI_idiom
estos directorios continen archivos .sh, .txt, .dat, .idiom que implementan las diferentes funciones del programa.
 
La instalación sobreescribira a cualquier version >= 0.4 de este script.
A partir de la version 0.4, el script se llama lcdgrab y no cdgrab como en 
versiones <= 0.3, se ha cambiado el nombre del script al descubrir que cdgrab
es una aplicación diferente implementada antes que este script.

El fichero de configuración de la versión 0.4 no será valido para
esta nueva versión.

Manejo:
-------

1.- Ejecute lcdgrab bajo la linea de comandos;
    (No se implementa ninguna gestión de usuarios).
    Esta gestión debera configurarse manualmente
    en el sistema. Usar con cautela en las opciones
    de escritura sobre dispositivos de bloque si
    la aplicación es usada como usuario root.
    Este usuario no tiene ninguna limitación sobre el
    sistema o la propia aplicación.

2.- Teclee el numero correspondiente a la opcion deseada;

3.- Introduzca los datos de configuración que le pida el programa;

Los datos que pedira el programa son:

	1.- Velocidad de grabación ;

    	Basta con teclear el numero , por ejemplo 16.
	No se acepta ningún valor de velocidad máxima.

	2.- Canal IDE de la grabadora ;

    	El canal donde esta conectada la grabadora.
    	Si no lo sabe ejecute la siguiente orden : cdrecord -scanbus

	Un ejemplo sobre valores del canal IDE son : 0,0,0 o 0,1,0

	Puede especificar tambien en este caso el dispositivo de bloque que
	corresponde con su grabadora bien /dev/hdd,/dev/hde etc,en lugar 
	del canal IDE.
	Segun su versión de cdrecord debe optar por un valor u otro.

3.- Etiqueta para el CD ;

    El nombre que quiera darle al CD , por ejemplo , fotografías

4.- Ruta de directorios ;

    La ruta de directorios de los datos a grabar , por ejemplo:
    /home/usuario/documentos

Manejo avanzado:
----------------

5.- Ejecución de lcdgrab con argumentos ;

    Es posible invocar a lcdgrab con argumentos, los argumentos hacen que
    linux-cdgrab cree automáticamente un fichero de configuración segun 
    los valores con los que se hizo la llamada al programa.
    Es una práctica util crear un breve script que invoque al programa
    pasando como argumentos la configuración que se suela usar en el sistema.
    Ha de tenerse en cuenta que el fichero de configuración de la aplicación
    sera sobreescrito o generado en disco en cada ejecución si usa esta
    práctica.
   
    Para listar los argumentos disponibles, ejecute : lcdgrab -h

6.- Modo de depuración ;
    
    El modo de depuración solo sirve para detectar errores de ejecución
    en linux-cdgrab.
    Si desea depurar el script escriba dep desde el menu principal.
    Si desea depurar un parche para linux-cdgrab escriba deppatch
    en el menu principal.

7.- Mostrar logotipo ASCII ;

    Escribiendo logo desde el menu principal se mostrara el
    logotipo ASCII de la aplicación.

8.- Consulta de la pagina man ;
    Consultar la pagina man de lcdgrab mendiante la orden:

    man lcdgrab
